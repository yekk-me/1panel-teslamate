## 截图建议

1. **界面要求**：
   - 截图时保持界面干净整洁
   - 突出显示操作按钮或重要区域
   - 隐藏敏感信息（如真实的用户名、密码、Token等）

2. **尺寸建议**：
   - 宽度不超过 1200px
   - 保持合适的长宽比，确保内容清晰可见

3. **格式要求**：
   - 使用 PNG 格式
   - 文件大小适中，不要过大

4. **命名规范**：
   - 使用小写字母
   - 单词之间用连字符（-）连接
   - 与手册中的引用保持一致

## 图片替换方法

将准备好的图片文件直接放入此目录，确保文件名与上述列表完全一致即可。
